讓工具用真的 AI 整理

沒有設定時，工具會用內建的模擬整理，流程一樣走得完。 設定之後，「請 AI 幫我整理」就會改用真的 AI。

還沒啟動 AI 代理

API Key 不能放在瀏覽器裡，所以需要一支小程式跑在你自己的電腦上替你保管。 打開終端機，在專案資料夾執行：

node tools/job-design/server/ai-proxy.mjs

看到「AI 代理已啟動」就成功了。這個視窗保持開著，然後回到這一頁重新整理。

哪一家的 Key 你申請得到就用哪一家，之後隨時可以換。

留空就用預設。若供應商更新了型號，直接改這裡即可。

填好後只會存在你電腦上的設定檔，不會存進瀏覽器，也不會進版控。

這支代理只綁定你自己的電腦（127.0.0.1），不對外開放。 未來要讓客戶也能用時，會改成放在雲端並由後端保管金鑰，屆時這一頁的設定方式不變。
